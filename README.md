# R💰 Ramalho Finance Pro V3

Aplicativo financeiro pessoal em HTML, CSS e JavaScript puro, com visual futurista e pronto para GitHub Pages.

## Funcionalidades

- Dashboard com receitas, despesas, saldo e taxa de economia
- Cadastro de receitas e despesas por categoria
- Histórico mensal persistente
- Fechamento de mês
- Despesas recorrentes automáticas
- Meta financeira com barra de progresso
- Insights automáticos
- Gráfico de gastos por categoria
- Gráfico de evolução mensal
- Exportação CSV
- Backup e restauração JSON
- Tema claro/escuro
- PWA instalável no celular
- Funciona offline após primeiro acesso

## Como publicar no GitHub Pages

1. Crie um repositório chamado `ramalho-finance`
2. Envie todos os arquivos desta pasta
3. Vá em `Settings > Pages`
4. Escolha `Deploy from a branch`
5. Selecione `main` e `/root`
6. Salve e aguarde o link ser gerado

## Como instalar no celular

Abra o site no Chrome do Android e toque em:

`Menu > Adicionar à tela inicial` ou `Instalar aplicativo`.
